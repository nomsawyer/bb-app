# Node.js環境構築
https://nodejs.org/ja/download/  
LTS版をダウンロード  
  
ターミナルでバージョン確認  
```  
node -v
//v12.18.2
```  
  
serverディレクトリへ移動し下記コマンド実行  
```  
npm install  
npm run start-node  
```  
  
localhost:3001でHello Worldが表示

# React起動  
appディレクトリへ移動し下記コマンド実行
```  
npm install  
npm start  
```  
  
localhost:3000でHello Worldが表示
